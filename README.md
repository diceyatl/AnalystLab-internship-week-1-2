# Data Cleaning & Exploratory Data Analysis (EDA)

## Project Overview
This project was completed as part of the AnalystLab Africa Data Analytics Internship (Batch B). It focuses on data cleaning, validation, and exploratory data analysis on two datasets:
- E-commerce Transactions Dataset
- Netflix Movies & TV Shows Dataset

## Objectives
- Clean and standardize raw datasets
- Handle missing values and duplicates
- Perform exploratory data analysis (EDA)
- Visualize key trends and patterns
- Generate insights from the data

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Steps
- Data loading and inspection
- Handling missing values
- Removing duplicates
- Standardizing formats
- Exploratory data analysis
- Data visualization

## Files in this repository
- Jupyter Notebook (.ipynb)
- Cleaned datasets (.csv)

## Key Insights
- Identified top-performing products in the e-commerce dataset
- Analyzed content trends in Netflix dataset
- Observed seasonal and categorical patterns in data
